# OtakuList

OtakuList is an API made for the 2019 UniCEUB web class.

Features:
- Login authentication.
- Account management.
- Anime and manga management and search.
- Anime and manga lists management.
- Forums management.
- Anime news.

## Made With
- Go-lang
- MongoDB
